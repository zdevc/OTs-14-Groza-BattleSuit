## 📦 安装方法

### 方法一：Steam 创意工坊（推荐）

1. 访问 [创意工坊页面](https://steamcommunity.com/sharedfiles/filedetails/?id=3694560137)
2. 点击 **"订阅"**
3. 游戏中启用模组

### 方法二：手动安装

> ⚠️ 如果游戏中显示白盒子（材质缺失），请用此方法。

1. [下载模组文件](https://github.com/zdevc/OTs-14-Groza-BattleSuit/releases/download/mod/OTs-14.Groza.BattleSuit.zip)
2. 解压到：`C:\Users\[用户名]\AppData\Roaming\JaggedAlliance3\Mods`
3. 游戏中启用

> 💡 `AppData` 是隐藏文件夹，需在文件夹选项中启用"显示隐藏文件"

## 📖 背景故事

OTs-14，代号「闪电」，艾莫号副官兼闪电小队队长。前格里芬精锐老兵，失忆后被指挥官从黄区废土捡回，在艾莫号重获新生。

她是指挥官最信赖的左膀右臂——战术指挥、人形协调、任务规划，样样精通。平时成熟稳重，为了帮指挥官省钱，一直舍不得安装味觉模块，做出来的饭菜……很有「闪电特色」。

近期大犬座之国出现神秘感染，闪电受命带领小队前往侦察。这不仅是任务，更是对她独立指挥能力的考验。

**闪电小队成员**: 寇尔芙、纳美西丝、克罗丽科、维普蕾
